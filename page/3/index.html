UIBezierPath

发表于 2016 08 20   |     |   阅读次数

UIBezierPath是什么

  1. 在iPad出现之前，大部分定义绘图只能够使用Core Graphics，因为UIKit并不能绘制任意形状。
  2. 在iOS3.2中添加了UIBezierPath更高级的API来绘制。实际上UIBezierPath是对CGPathRef的封装。
  3. UIKit依然缺乏对渐变、阴影等高级特性的支持。但UIKit却可以非常方便的实现大部分常见的自定义绘制。
